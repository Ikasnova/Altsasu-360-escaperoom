Fase 5: Exportación y Distribución

Exportación y Distribución del Proyecto

Exportar como paquete SCORM

Para alojar el proyecto en Moodle u otras plataformas LMS:

  1. "Archivo > Exportar > SCORM 1.2" o "SCORM 2004" (verificad compatibilidad con vuestra plataforma)
  2. Guardad el archivo ZIP generado
  3. Subid el paquete SCORM a vuestra aula virtual (Moodle, Aula Ikas, etc.)
  4. Configurad las opciones de seguimiento y calificación

Exportar como sitio web autónomo

Para distribución sin LMS:

  1. "Archivo > Exportar > Sitio Web".
  2. eXeLearning generará una carpeta con todos los archivos HTML.
  3. Compressed la carpeta en ZIP y compartidla con el centro alemán.
  4. Podéis alojarla en Google Drive, Dropbox o servidor web para acceso online.

Compartir con el centro alemán

Estableced canales de comunicación y distribución:

  • eTwinning: Registrad el proyecto en la plataforma eTwinning para proyectos colaborativos europeos.
  • Google Classroom o Teams: Cread un espacio conjunto donde ambos grupos de estudiantes accedan al escape room.
  • Servidor compartido: Alojad el proyecto en un servidor accesible desde ambos países.
  • Carpetas compartidas: Usad OneDrive, Google Drive o similares para compartir recursos.
